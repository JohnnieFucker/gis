# GIS 区域地图标注系统

一个简单易用的区域地图标注工具，支持在地图上输入经纬度并标注点位。

## 功能特性

- 🗺️ 区域地图显示（可自定义地图中心位置和缩放级别）
- 📍 经纬度输入标注功能
- 🏷️ 支持为标注点添加名称
- 📋 标注点列表管理
- 🔍 快速定位到指定标注点
- 🗑️ 支持删除单个或清除所有标注点
- 💻 响应式设计，支持移动端

## 快速开始

### 方法一：使用 Python 简单服务器（推荐）

1. 在项目目录下运行：

```bash
# Python 3
python3 -m http.server 8000

# 或 Python 2
python -m SimpleHTTPServer 8000
```

2. 打开浏览器访问：`http://localhost:8000`

### 方法二：使用 Node.js 服务器

```bash
# 安装 http-server（如果未安装）
npm install -g http-server

# 在项目目录下运行
http-server -p 8000
```

### 方法三：直接打开文件

某些浏览器可能由于安全限制不允许直接打开本地 HTML 文件，建议使用上述服务器方法。

## 配置说明

### 修改地图中心位置

编辑 `map.js` 文件中的 `MAP_CONFIG` 对象：

```javascript
const MAP_CONFIG = {
    // 修改为您公司的坐标（纬度, 经度）
    center: [39.9042, 116.4074],  // 示例：北京天安门
    // 初始缩放级别（1-18，数值越大越近）
    zoom: 13,
    // 最小缩放级别，限制不能缩得太远
    minZoom: 10,
    // 最大缩放级别
    maxZoom: 18
};
```

### 如何获取公司坐标？

1. 使用在线地图工具（如高德地图、百度地图、Google Maps）
2. 在地图上找到公司位置
3. 右键点击获取经纬度坐标
4. 格式：`[纬度, 经度]`，例如：`[39.9042, 116.4074]`

## 使用指南

### 添加标注点

1. 在左侧控制面板的"添加标注点"区域
2. 输入纬度（Latitude）和经度（Longitude）
3. 可选：输入标注名称
4. 点击"添加标注"按钮
5. 标注点会自动显示在地图上，并弹出信息窗口

### 管理标注点

- **查看列表**：所有标注点会显示在左侧"标注列表"区域
- **定位**：点击标注列表中的"定位"按钮，地图会自动定位到该点
- **删除**：点击"删除"按钮可移除单个标注点
- **清除所有**：点击"清除所有标注"按钮可一次性删除所有标注点

### 快捷操作

- 点击地图上的任意位置，会自动填充该位置的经纬度到输入框

## 技术栈

- **Leaflet.js** - 开源地图库

## 注意事项

1. 默认使用 OpenStreetMap 作为地图源，如果需要使用其他地图服务（如高德、百度等），需要修改 `map.js` 中的地图瓦片URL
2. 经纬度范围：纬度 -90 到 90，经度 -180 到 180
3. 刷新页面后，标注点会丢失（如需持久化存储，可以考虑使用 localStorage 或后端数据库）
